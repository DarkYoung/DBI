## 使用方法：

- [x] 1、运行jar包
> * pwd：DBI（重要） 
> * 命令：`java -jar lib/DBI.jar` 
> * 根据提示选择需要导入的数据源 
> * 开始导入…… 

- [x] 2、使用idea打开项目 
> * 运行MainApp.java 
> * 选择数据源 
> * 开始导入…… 

---

## 配置：
### jdbc.properties：配置数据库连接（MySQL、csv、sqlite）
### sql.properties：sql语句（创建数据库）
### insert_dup_log：导入失败的重复数据（SQL语句）
### insert_log：成功导入的数据（SQL语句）
